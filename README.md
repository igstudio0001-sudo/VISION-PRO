After a long period of development, I want to show you a project that I am currently working on.

This is a system for counting, analyzing, and monitoring, as well as for reporting and is designed to automate the input control process in enterprises of any size.

The main goal is to reduce data processing time as much as possible, minimize the human factor, and provide employees with a convenient tool for making decisions. What can the system do?

Counting products by photo.
Labeling each detected object.
Using color categories to separate information.
Automatically calculating the total weight of a batch.
Generating a detailed report after analysis.
Flexibly adjusting parameters to meet the specific requirements of an enterprise.
How does it work?

The user uploads an image of the product.

Next, the system automatically:

assigns a number to each product;
calculates the quantity;
calculates the total weight;
generates a ready-made report with the analysis results.
The entire process takes less time and is significantly faster than manual counting.

speed up the input control process;
